# python-backend-course-fall-2022

## О проекте

Пусть это будет сервис доставки еды с интерфейсом в Телеграм-боте.

##### Сервис умеет:

- Делать заказ

- Добавлять/редактировать/удалять блюда

##### В дальнейшем сервис будут реализованы:

- Аутентификация пользователя

- Аутентификация для бизнеса

- Панель управления для бизнеса

- Интерфейс в виде Telegram-бота
